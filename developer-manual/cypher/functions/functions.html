술어 함수

이 함수는 주어진 인수에 대해 true 또는 false를 반환합니다.

기능설명
all()조건자(predicate)가 리스트의 모든 요소에 적용되는지 여부를 테스트합니다.
any()술어가 목록에있는 하나 이상의 요소에 대해 보유하는지 여부를 테스트합니다.
exists()그래프와 패턴이 일치하거나 노드, 관계 또는 맵에 속성이 있는 경우 true를 반환합니다.
none()술어가 리스트에 요소가 없는 경우 true를 리턴합니다.
single()술어가 리스트의 요소 중 하나에 대해 보유하는 경우 true를 리턴합니다.
스칼라 함수

이 함수는 단일 값을 반환합니다.

기능설명
coalesce()전달 된 표현식 목록의 첫 번째 null 값을 반환합니다.
endNode()관계의 마지막 노드를 반환합니다.
head()리스트의 첫번째 요소를 반환합니다.
id()관계 또는 노드의 ID를 반환합니다.
last()리스트의 마지막 요소를 반환합니다.
length()패스의 길이를 반환합니다.
properties()인수가 노드 또는 관계인 경우 반환 된 맵은 해당 속성의 맵입니다.
size()목록에있는 항목의 수를 반환합니다.
패턴표현의 크기패턴 표현식과 일치하는 하위 그래프의 수를 반환합니다.
문자열의 크기문자열의 크기를 반환합니다.
startNode()관계의 첫번째 노드를 돌려줍니다.
timestamp()현재 시간과 UTC 1970 년 1월 1일 자정 사이의 밀리 초 단위 차이를 반환합니다.
toBoolean()인수를 부울로 변환하고 결과를 반환합니다.
toFloat()인수를 float 로 변환하고 결과를 반환합니다.
toInteger()인수를 정수로 변환하고 결과를 반환합니다.
type()관계 타입의 표현을 반환합니다.
집계 함수

이 함수는 여러 값을 인수로 사용하여 집계 된 값을 계산하여 반환합니다.

기능설명
avg()숫자 열의 평균을 구합니다.
collect()수집 된 모든 값을 포함하는 목록을 반환합니다.
count()행수를 돌려줍니다.
max()숫자 열에서 가장 큰 값을 반환합니다.
min()숫자 열에서 가장 작은 값을 반환합니다.
percentileCont()선형 보간법을 사용하여 그룹에 대해 주어진 값의 백분위 수를 반환합니다.
percentileDisc()반올림 메서드를 사용하여 그룹에 대해 주어진 백분위 수에 가장 가까운 값을 반환합니다.
stDev()모집단 샘플에 대한 그룹에 대해 주어진 값의 표준 편차를 구합니다.
stDevP()전체 채우기에 대해 그룹에 대해 주어진 값의 표준 편차를 구합니다.
sum()수치 열의 합계를 돌려줍니다.
리스트 함수

이 함수는 다른 값 목록을 반환합니다. 리스트의 더 자세한 내용과 예제는 3.2.8 절 "리스트" 에서 찾을 수 있습니다.

기능설명
extract()단일 속성 또는 노드 또는 관계 목록에서 함수 값을 반환합니다.
filter()술어를 준수하는 목록의 모든 요소를 ​​반환합니다.
keys()노드, 관계 또는 맵의 속성 이름에 대한 문자열 표현 목록을 반환합니다.
labels()노드에 첨부 된 레이블에 대한 문자열 표현 목록을 반환합니다.
nodes()경로의 모든 노드를 반환합니다.
range()범위 내의 수치를 돌려줍니다.
reduce()목록의 개별 요소에 대해 표현식을 실행하여 표현식의 결과를 누적기에 저장합니다.
relationships()경로의 모든 관계를 반환합니다.
tail()리스트의 최초의 요소 이외의 모든 것을 돌려줍니다.
수학 함수

이 함수는 모두 숫자 식에만 적용되며 다른 값에 사용하면 오류가 반환됩니다.

수치 함수
기능설명
abs()숫자의 절대 값을 반환합니다.
ceil()인수보다 크거나 같은 가장 작은 정수를 구합니다.
floor()표현식보다 작거나 같은 가장 큰 정수를 반환합니다.
rand()0 이상 1 이하의 범위의 난수를 돌려줍니다.
round()가장 가까운 정수로 반올림 한 숫자 식을 반환합니다.
sign()숫자의 부호를 반환합니다. 표현식이 0이면 0, 임의의 음수는 -1 , 양수이면 1 을 반환합니다.
로그 함수
기능설명
e()자연 대수의 밑, e를 구합니다.
exp()e^n 반환합니다. 여기서 e 는 자연 대수의 밑이며, n 은 인수 표현식의 값입니다.
log()식의 자연 대수를 반환합니다.
log10()식의 공통 대수 (밑 10)를 반환합니다.
sqrt()숫자의 제곱근을 반환합니다.
삼각 함수

달리 지정되지 않는 한 모든 삼각 함수는 라디안에서 작동합니다.

기능설명
acos()표현식의 아크 코사인(arccosine)을 반환합니다.
asin()표현식의 아크 사인(arcsine)을 반환합니다.
atan()표현식의 아크 탄젠트(arctangent)를 반환합니다.
atan2()좌표 세트의 아크 탄젠트 2(arctangent2)를 돌려줍니다.
cos()표현식의 코사인(cos)을 반환합니다.
cot()표현식의 코탄젠트(cotangent)를 반환합니다.
degrees()라디안을 도 단위로 변환합니다.
haversin()표현식의 절반을 반환합니다.
pi()수학 상수 pi를 반환합니다.
radians()도를 라디안으로 변환합니다.
sin()표현식에서 사인(sin)을 반환합니다.
tan()표현식에서 탄젠트(tan)을 반환합니다.
문자열 함수

이 함수는 문자열을 조작하거나 다른 값의 문자열 표현을 작성하는 데 사용됩니다.

기능설명
left()원래 문자열의 왼쪽 n 문자를 포함하는 문자열을 반환합니다.
lTrim()왼쪽에서 공백을 제거한 원래 문자열을 반환합니다.
replace()검색 문자열이 대체 문자열로 대체 된 문자열을 반환하여 모든 항목을 대체합니다.
reverse()원 문자열의 반대열을 반환합니다.
right()원래 문자열의 오른쪽 n 문자를 포함하는 문자열을 반환합니다.
rTrim()오른쪽에서 공백을 제거한 원래 문자열을 반환합니다.
split()분할 패턴으로 구분 된 문자열의 시퀀스를 반환합니다.
substring()0부터 시작하는 인덱스 start 및 length를 사용하여 원본의 하위 문자열을 반환합니다.
toLower()소문자로 된 원래 문자열을 반환합니다.
toString()인수를 문자열로 변환합니다.
toUpper()대문자로 된 원래 문자열을 반환합니다.
trim()양쪽에서 공백을 제거한 원래 문자열을 반환합니다.
공간 함수

이 함수는 2D 좌표계에서 점을 지정하고 두 점 사이의 측지선 거리를 계산하는 데 사용됩니다.

기능설명
distance()지정된 두 점 사이의 측지선 거리를 나타내는 부동 소수점 숫자를 반환합니다.
point()두 개의 좌표 값이있는 점 객체를 반환합니다.
